# BOSL2 Quickstart

[Launch this project in the Varcad IDE](https://varcad.io/varcad-io/bosl2-quickstart)

A minimal introduction to using BOSL2 in a Varcad OpenSCAD project.

## What this demonstrates

- linking BOSL2 through Varcad's library manager
- including BOSL2's standard module collection
- creating a rounded cuboid
- changing dimensions interactively with widgets

## Library import

```scad
include <@bosl2/std.scad>;
```

Open the project in Varcad, adjust the width, depth, height, and rounding controls, and render the result.

Upstream documentation: [BOSL2 Wiki](https://github.com/BelfrySCAD/BOSL2/wiki)
